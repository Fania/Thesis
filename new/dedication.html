Acknowledgments

This website has been created based on the original LaTeX source files for the corrected thesis. Each chapter has been individually converted to html using pandoc and much love1.

Most of it translated gracefully into html apart from figures and spiral poems (which are shown as images), and references (which are listed at the end of each chapter).
